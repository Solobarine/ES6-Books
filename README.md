# Project Name

> "Awesome books" is a simple website that displays a list of books and allows you to add and remove books from that list modified to follow ES6 rules.


## Built With

- JavaScript
- HTML5
- CSS3

## How to use

1. Clone the project using "git clone https://github.com/solobarine/ES6-Books"
2. Run the projec on a server (example: [live server on vscode](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer))
